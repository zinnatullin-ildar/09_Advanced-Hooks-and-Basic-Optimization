React.cloneElement() клонирует и возвращает новый React-элемент. Первым параметром он
принимает элемент, вторым config и третьим потомков (опционально):

React.cloneElement(element, config, ...children);
где config может содержать все новые пропсы, key и ref

Полученный таким способом элемент будет иметь пропсы исходного элемента (element) и также
новые пропсы из config. Если в config совпадет название пропса, который уже есть у element,
то применится проп из config.
Переданные key и ref из исходного элемента (element) будут сохранены, если в config они не были переданы. 
Новые дочерние элементы (children) заменят существующие (у element).
